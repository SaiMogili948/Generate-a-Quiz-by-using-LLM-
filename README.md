# Wiki Quiz App

## Objective
Generate quizzes automatically from Wikipedia articles using LLMs.

 ## Tech Stack
 Backend: FastAPI
 Frontend: HTML + JavaScript
 Database: PostgreSQL
 Scraping: BeautifulSoup
 LLM: Gemini (via LangChain)

## Setup Instructions

### Backend
bash
pip install -r requirements.txt
uvicorn main:app --reload
